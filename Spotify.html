    Spotify Help

    Display information from Spotify's API

    spotify

    Description: Spotify commands

    Usage: ,spotify

    spotify toptracks

    Description: List your top tracks on spotify

    Usage: ,spotify toptracks

    spotify artist

    Description: View Spotify Artist info

    Usage: ,spotify artist

    spotify artist albums

    Description: View an artists albums

    <to_follow> The artis links or URI's you want to view the albums of

    Usage: ,spotify artist albums

    spotify artist follow

    Description: Add an artist to your spotify library

    <to_follow> The song links or URI's you want to have removed

    Usage: ,spotify artist follow

    spotify previous

    Description: Skips to the previous track in queue on Spotify

    Usage: ,spotify previous

    spotify playlist

    Description: View Spotify Playlists

    Usage: ,spotify playlist

    spotify playlist remove

    Description: Remove 1 (or more) tracks to a spotify playlist

    <name> The name of playlist you want to remove songs from <to_remove> The song links or URI's you want to have removed

    Usage: ,spotify playlist remove

    spotify playlist view

    Description: View details about your spotify playlists

    If this command is done in DM with the bot it will show private playlists otherwise this will not display private playlists unless showprivate has been toggled on.

    Usage: ,spotify playlist view

    spotify playlist list

    Description: List your Spotify Playlists

    If this command is done in DM with the bot it will show private playlists otherwise this will not display private playlists unless showprivate has been toggled on.

    Usage: ,spotify playlist list

    spotify playlist create

    Description: Create a Spotify Playlist

    <name> The name of the newly created playlist [public] Wheter or not the playlist should be public, defaults to False. [description] The description of the playlist you're making.

    Usage: ,spotify playlist create

    spotify playlist featured

    Description: List your Spotify featured Playlists

    Usage: ,spotify playlist featured

    spotify playlist follow

    Description: Add a playlist to your spotify library

    [public] Whether or not the followed playlist should be public after <to_follow> The song links or URI's you want to have removed

    Usage: ,spotify playlist follow

    spotify playlist add

    Description: Add 1 (or more) tracks to a spotify playlist

    <name> The name of playlist you want to add songs to <to_add> The song links or URI's you want to add

    Usage: ,spotify playlist add

    spotify repeat

    Description: Repeats your current song on spotify

    ["off"|"track"|"context"] must accept one of off, track, or context.

    Usage: ,spotify repeat

    spotify shuffle

    Description: Shuffles your current song list

    <state> either true or false. Not providing this will toggle the current setting.

    Usage: ,spotify shuffle

    spotify device

    Description: Spotify device commands

    Usage: ,spotify device

    spotify device default

    Description: Set your default device to attempt to start playing new tracks on if you aren't currently listening to Spotify.

    <device_name> The name of the device you want to switch to.

    Usage: ,spotify device default

    spotify device transfer

    Description: Change the currently playing spotify device

    <device_name> The name of the device you want to switch to.

    Usage: ,spotify device transfer

    spotify device list

    Description: List all available devices for Spotify

    Usage: ,spotify device list

    spotify queue

    Description: Queue a song to play next in Spotify

    <songs> is one or more spotify URL or URI leading to a single track that will be added to your current queue

    Usage: ,spotify queue

    spotify play

    Description: Play a track, playlist, or album on Spotify

    <url_or_playlist_name> can be multiple spotify track URL or URI or a single album or playlist link

    if something other than a spotify URL or URI is provided the bot will search through your playlists and start playing the playlist with the closest matching name

    Usage: ,spotify play

    spotify recent

    Description: Displays your most recently played songs on Spotify

    Usage: ,spotify recent

    spotify seek

    Description: Seek to a specific point in the current song

    <seconds> Accepts seconds or a value formatted like 00:00:00 (hh:mm:ss) or 00:00 (mm:ss).

    Usage: ,spotify seek

    spotify volume

    Description: Set your spotify volume percentage

    <volume> a number between 0 and 100 for volume percentage.

    Usage: ,spotify volume

    spotify new

    Description: List new releases on Spotify

    Usage: ,spotify new

    spotify topartists

    Description: List your top artists on spotify

    Usage: ,spotify topartists

    spotify recommendations

    Description: Get Spotify Recommendations

    <recommendations> Requires at least 1 of the following matching objects: - genre: Must be a valid genre type. Do [p]spotify genres to see what's available. - tracks: Any spotify URL or URI leading to tracks will be added to the seed - artists: Any spotify URL or URI leading to artists will be added to the seed

    The following target parameters also exist and must include some additional value: acousticness:, danceability:, energy:, instrumentalness:, liveness:, popularity:, speechiness:, and/or valence: + a value from 0-100

    key: + A value from 0-11 representing Pitch Class notation loudness: + A value from -60 to 0 represending dB mode: + either major or minor tempo: + the tempo in BPM time_signature: + the measure of bars e.g. 3 for 3/4 or 6/8 duration_ms: the duration target of the tracks

    e.g. [p]spotify recommendations genre: edm electronic valence: 100 mode: major

    Usage: ,spotify recommendations

    spotify me

    Description: Shows your current Spotify Settings

    Usage: ,spotify me

    spotify next

    Description: Skips to the next track in queue on Spotify

    Usage: ,spotify next

    spotify pause

    Description: Pauses spotify for you

    Usage: ,spotify pause

    spotify resume

    Description: Resumes spotify for you

    Usage: ,spotify resume

    spotify now

    Description: Displays your currently played spotify song

    [member] Optional discord member to show their current spotify status if they're displaying it on Discord.

    Usage: ,spotify now

    spotify share

    Description: Tell the bot to play the users current song in their current voice channel

    Usage: ,spotify share

    spotify search

    Description: Search Spotify for things to play

    [detailed=False] Show detailed information for individual tracks. [search_type=track] The search type, available options are: - track(s) - artist(s) - album(s) - playlist(s) - show(s) - episode(s) <query> What you want to search for.

    Usage: ,spotify search

    spotify set

    Description: Setup Spotify cog

    Usage: ,spotify set

    spotify set showemojis

    Description: Show information about the currently set emoji pack

    Usage: ,spotify set showemojis

    spotify set resetemojis

    Description: Resets the bot to use the default emojis

    Usage: ,spotify set resetemojis

    spotify set currentscope

    Description: View the current scopes being requested

    Usage: ,spotify set currentscope

    spotify set creds

    Description: Instructions to set the Spotify API tokens.

    Usage: ,spotify set creds

    spotify set menutimeout

    Description: Set the timeout time for spotify menus

    <timeout> The time until the menu will timeout. This does not affect interacting with the menu. Note: This has a maximum of 10 minutes and a minimum of 30 seconds.

    Usage: ,spotify set menutimeout

    spotify set clearreactions

    Description: Set whether or not to clear reactions after sending the message

    Note: the bot requires manage messages for this to work

    Usage: ,spotify set clearreactions

    spotify set forgetme

    Description: Forget all your spotify settings and credentials on the bot

    Usage: ,spotify set forgetme

    spotify set scope

    Description: Set customized scope for what you want your bot to allow

    Available options are: user-read-private user-top-read user-read-recently-played user-follow-read user-library-read user-read-currently-playing user-read-playback-state user-read-playback-position playlist-read-collaborative playlist-read-private user-follow-modify user-library-modify user-modify-playback-state playlist-modify-public playlist-modify-private

    You can find more information here: https://developer.spotify.com/documentation/general/guides/scopes/

    Usage: ,spotify set scope

    spotify set deletemessage

    Description: Set whether or not to delete the spotify message after timing out

    Usage: ,spotify set deletemessage

    spotify set showsettings

    Description: Show settings for menu timeouts

    Usage: ,spotify set showsettings

    spotify set emojis

    Description: Change the emojis used by the bot for various actions

    [new_emojis] Is a space or comma separated list of name followed by emoji for example [p]spotify set emojis playpause will then replace usage with the emoji.

    This command also accepts a .yaml file containing emojis if using custom emojis they must be in the format a:name:12345 or :name:12345

    Example: yaml playpause: pause: repeat: repeatone: next: previous: like: fastforward: rewind: volume_down: volume_up: volume_mute: playall: shuffle: back_left: forward_right: play: queue:

    Usage: ,spotify set emojis

    spotify set showprivate

    Description: Set whether or not to show private playlists

    This will also display your spotify username and a link to your profile if you use [p]spotify me command in public channels.

    Usage: ,spotify set showprivate

    spotify genres

    Description: Display all available genres for the recommendations

    Usage: ,spotify genres
